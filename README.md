# REALTOREPROS CRM

Enterprise real estate CRM and realtor management system for RealtyPros Investment Global Ltd.

## Run locally

```bash
npm install
npm run dev
```

Open the web application:

```text
http://localhost:3000
```

Open the mobile application workspace:

```text
http://localhost:3000/mobile
```

## Included product areas

- Executive Dashboard
- CSMO Sales Management
- COO Operations
- REPROS Realtor Management
- Realtor Learning Hub
- Customer Care System
- Product Management
- Front Desk Management
- Media Management
- Reports Center
- AI & Automation Engine
- Administration & Access Control
- Audit Logs
- Android/iOS mobile workspace

## Architecture direction

The interface is prepared for a Supabase PostgreSQL backend with Row Level Security, REST APIs, GraphQL support, secure file storage, authentication, notifications, audit logs, and role-based access control.
